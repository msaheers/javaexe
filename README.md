GUI Calculator (Java Swing)
 Project Overview

This is a GUI-based calculator built using Java and Swing.

The calculator supports:

Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Decimal numbers
Full expression evaluation (e.g., 7+7-9*45/8)
Blinking cursor effect in display
Clear (A/C) button

The goal of this project was to:

Practice Java GUI development
Understand event handling
Learn layout positioning
Implement a custom mathematical expression parser
Avoid using deprecated ScriptEngine
Improve logical thinking and recursion understanding

Technologies Used
Java
Swing (GUI Framework)
AWT Event Handling
Custom Recursive Expression Parser
